# Detecting_sentiment_of_quote_from_images
This project is a solution to problem statement by HackerEarth Machine Learning challenge: Love is love

## Problem Statement
https://www.hackerearth.com/challenges/competitive/hackerearth-machine-learning-challenge-pride-month-edition/

In this problem statement first we have extract the text data from image file.
For this I use keras ocr. 
Then Detect the sentiment of text using NLP.
